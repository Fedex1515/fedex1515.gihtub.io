Home


Homepage with ip list. You can create different profiles for you networks and keep track of your network device as well run custom application passing different parameters as arguments for thirdy part applications.


Device View


Detailed description for each device in your network. This view show a more detailed view of the device selected in the homepage.


Settings


In the page settings you can add up to 16 thirdy parts appllication running on a specific ip inserted in the home page.


Custom scripts


You can write custom sripts to run according to the current profile for example to add or remove custom routes. If you need to run these script as admin you must tick "Run as administrator" in the property of the exe file.
